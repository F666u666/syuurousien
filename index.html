1. 概要

就労移行IT支援の強み（ITスクール × メンタルケア × 診断 × 転職の一気通貫）を、「ある女性が社会復帰するまでの記録」というドキュメンタリー連載（プロセスエコノミー）で可視化する。完成された専門家ではなく“もがいて前に進む当事者”を見せることで共感を生み、潜在層を集めて無料面談まで運ぶ。主役は実在の本人・顔出しなし。診断軸は「適応障害 ×（根っこの）ADHD」の二段構造。売りは「人間関係で疲れた人が、“人と関わらず在宅で働ける”ようになる」こと。媒体はInstagram＋Threads、全国オンライン対応、オーガニック運用。最重要KPIは面談予約数。

2. 目的・ゴール

  • 最終ゴール：無料面談の予約獲得
  • SNSの役割：「自分、もしかして…」で止まっている人を、面談の一歩目まで運ぶ
  • フォロワー数は手段。煽れない領域なので“数”を追わず、保存率・プロフィール遷移・面談数を追う

3. ターゲット

主軸セグメント：25〜40歳（コア27前後）。うつ・適応障害を起点に「働きたいのに働けない自分」を責める層。根っこにADHD（発達）があり二次障害でうつ・適応障害を発症した型、HSP・「生きづらさ」自認、在宅志向・ネット好き（IT親和性が高い）も重なる。

見ている人＝3タイプ

A 潜在本人予備軍。未受診。「これ、今の私だ」。最大数・遠い → 物語で集めて時間をかけて育てる
B 渦中診断済み・休職/離職中。あかりと同じ地点。最も面談に近い ＝刈り取りの主軸
C 家族「うちの子／夫がこれかも」。保存して本人に見せる・拡散の裏方

数を作るのはA、面談に変えるのはB、拡散の裏方がC。／ やらない：統合失調症（自分から検索して動く層でない）・知的・身体（IT×メンタルの強みと噛み合わない）。

4. ペルソナ（主役）

あかり・27歳・女性／地方都市で一人暮らし

診断（二段構造）：「適応障害（うつ状態）」、そして根っこに「ADHD」。

ADHD（根っこ・生まれつき）→ 特性を気合いで我慢し続ける → 適応障害（きっかけ＝仕事・環境）→ うつ状態（沈みが長引く＝二次障害）

潰れた経緯：子どもの頃から“何かおかしい”。ケアレスミス・要領の悪さ・忘れ物で「やる気ある?」と言われ続けた（＝ADHD特性）。気合いでカバーして無理を続け、約2年で限界 → ある朝、体が動かなくなり受診 → “適応障害、そしてADHD”と判明 → 休職、そして退職。離職の決め手は「人間関係・電話恐怖」（雑談が苦痛／電話が鳴るだけで動悸）。

現在地：回復期。貯金を切り崩しながら、就労移行のITスクールに通い始めたばかり（プログラミング未経験）。ADHDの過集中がプログラミングで才能として開花していく＝物語後半の希望。

1日（今）7:00 起きるのに1時間／13:00 通所（不安だけど少し楽しい）／18:00 小さな達成感、でも夕方から不安／23:00 布団でスマホ、スクロールが止まらない
夜中の検索語「適応障害 復職 こわい」「働くの 向いてない」「在宅 仕事 人と話さない」「HSP 仕事 続かない」「25歳 急に 生きづらい」
動けない壁お金の不安／また潰れる恐怖／“障害者枠”への抵抗／周りの目／電話・対人への恐怖
家計のリアル休職中は傷病手当金＝給料の約2/3、退職後は貯金切り崩し。過去の衝動買いでリボ残高あり。それでも“推し活”にだけはお金を使う（唯一の生きる支え）
接触夜のInstagram・Threads（23〜25時がピーク）

表面の欲求：「もう一度働けるようになりたい」
本当の欲求：「“ちゃんとした自分”に戻って、自分を許したい。もう一度、安心して眠りたい」（物語が刺すのはこちら。就職はゴールでなく、自分を許すための手段）

※主役は実在の本人。顔出しなし（後ろ姿・手元・声・テロップで構成）。潰れた原因・生活状況は本人の事実に合わせて調整。

5. 悩み（精神科／メンタルケア視点・6層）

表面の「働けない」だけでは薄い。当事者の悩みは6層で積み重なる。連載で①〜⑥を体験させ、単発で①〜⑥をピンポイントに解く。

  • ① 症状 — 朝起きられない・頭が回らない・集中できない・眠れない・何も楽しくない（怠けでなく症状）
  • ② 認知の歪み — 白黒思考／べき思考／過度の一般化／自己関連づけ。自分を責める“クセ”自体が病気の症状
  • ③ 自己肯定感の崩壊 — 「働けない自分に価値がない」「迷惑な存在」。元できる子ほど落差で深く苦しむ
  • ④ 対人・孤立（二次被害） — 打ち明けられない／理解されない孤独／「甘え」と言われる傷／家族との軋轢
  • ⑤ 回復過程“特有”の悩み（差別化の肝） — 回復期の波／「治りかけが一番しんどい」／焦り／休む罪悪感
  • ⑥ 将来・社会復帰 — また潰れる恐怖／ブランク／障害者雇用への偏見／お金／カミングアウト不安

6. 物語の設計：応援される4原理

プロセスエコノミーの核。「過程」を見せるだけでは伸びない。人を動かすのはこの4つ。

① 完璧な人は応援されない

もがく人が応援される。だから過程を“途中から全部”見せる。結果だけ見せると他人事になる

② 小さな勝利を実況する

「朝起きられた」「散歩できた」。健康な人には何でもない一歩が、当事者には大事件

③ 波（後退）を隠さない

回復期は医学的に“揺れる”。正直に見せると信頼される。右肩上がりの嘘はバレて冷める

④ 助けられる側 → 助ける側へ

後半で支える側に回った瞬間、フォロワーが“ファン”から“当事者コミュニティ”に変わる

「うつは甘え」への向き合い方：怒って反論しない。「うつは意志や甘えじゃなく、脳のエネルギー切れ。気合いで治らないから治療がある」と“事実で静かに返す”。敵を作らず、当事者を全員味方にする。

7. 回復ロードマップ（社会復帰の7段階）

これは「物語（ドキュメンタリー）」であり、実際に1年かけて運用するものではない。1投稿ごとに“少しだけ前進する様子”と“その時の葛藤”を描いて、物語として進めていく。下の臨床経過は、各話のリアリティを支える背景設定として参照する（実際の制作スケジュールは13章を参照）。

精神科の標準的な回復過程（急性期→回復期→再発予防）と就労移行の流れに沿う。回復は直線ではなく、行きつ戻りつする。連載の7フェーズと一致。

段階臨床メモ（物語の背景）経過感の目安投稿の狙い・応援スイッチ
① 潜在・気づき受診前。不調の自覚—「実は〇〇でした」で共感（同じだ）
② 診断名前がつく転機—弱さの開示で信頼が生まれる
③ 休養・急性期休養と服薬が最優先物語上 約1〜3ヶ月「休むのも治療」罪悪感の解放
④ 回復期意欲が戻るが“波”が激しい物語上 約4〜6ヶ月小さな勝利＋隠さない後退
⑤ 自己理解原因の言語化・障害受容物語上 回復期〜“参考になる”＝保存される
⑥ 就労移行・ITスキル習得・模擬就労物語上 数ヶ月変化の可視化（最も伸びる）
⑦ 就職・社会復帰就職＋定着・再発予防物語上 終盤大団円＋他者貢献でコミュニティ化

1投稿の作り方：各話は「前回からの小さな一歩」＋「その一歩に伴う迷い・葛藤」をワンセットで描く。大きく進めず、少しずつ。これが“見届けたくなる”連載のリズムを作る。

9. 連載 企画一覧（全39本）

各話＝そのまま使える“素っ気ないのにテンションの効いた”冒頭フック（落差・ループ・感情・問い・情景のどれかを1個仕込む）。節目（診断・退職・通所初日・内定）は二度撮れないため作り溜め。

連載のリズム＝8ビート：発覚 → 共感 → 回復期1 → 共感 → 回復期2 → 社会復帰手前 → 回復 → 復帰。進む回と“共感（あるある）で刺す回”を交互に挟み、離脱を防ぐ。

Phase 1潜在・気づき（診断前）
  1. 去年までバリバリ働いてた私が、今、無職です
  2. 「頑張れば治る」って、本気で信じてた頃の話
  3. 満員電車で、急に涙が止まらなくなった
  4. 夜中2時、「涙 理由なく」で検索してる自分がいた
  5. 心療内科の予約ボタン、2週間押せなかった
Phase 2診断
  1. 精神科の待合室で、私だけ場違いな気がした
  2. 「適応障害です」って言われて、正直ちょっとホッとした
  3. 「よく我慢したね」のひと言で、涙が止まらなかった
  4. 会社に診断書を出した日、手が震えてた
  5. 一番わかってほしい人に、「甘えでしょ」って言われた
Phase 3休養・急性期
  1. 上司に「辞めます」って電話した日、声が震えてた
  2. 退職届を出した瞬間、ホッとして、こわくなった
  3. 一日中寝てるだけの自分が、一番きらいだった
  4. 薬を飲み始めて変わったこと。正直、こわかったことも
  5. はじめてのカウンセリング。何を話せばいいかも分からなかった
  6. 通帳の残高が減っていくのを見るのが、こわい
Phase 4回復期（波）
  1. はじめて外に散歩に出られた日。それだけで泣きそうだった
  2. 「朝起きる」を取り戻すのに、私がやった3つのこと
  3. 治ったと思った次の日、また動けなくなった
  4. 気分を記録し始めたら、自分の波が見えてきた
  5. 友達からのLINE、こわくて3日開けなかった
Phase 5自己理解・動き出す
  1. なんで私は壊れたのか。やっと、理由が分かった
  2. 「自分の取扱説明書」を作ってみたら、少しラクになった
  3. 障害者手帳をとるか、3日間ずっと悩んでた
  4. 「会社に行かなくても働ける」って、はじめて知った
Phase 6就労移行・スクール（就職に向けて動く）
  1. 無職の私が、「就労移行」ってやつを調べ始めた
  2. スクールの見学に行った日、久しぶりに外の空気を吸った
  3. 通所初日。緊張で、前の夜ほとんど眠れなかった
  4. 未経験の私が、はじめて自分でコードを動かした
  5. 「私だけじゃなかった」。同じ境遇の人に、はじめて会えた
  6. 支援員さんと、「いつか就職する」目標を立てた
  7. 模擬面接、ボロボロだった。それでも笑われなかった
  8. はじめての職場実習。行く前、足がすくんだ
  9. 履歴書の「空白期間」、どう書けばいいか分からなかった
Phase 7就職・社会復帰
  1. 面接で病気のこと、言うか言わないか、ギリギリまで迷った
  2. 無職だった私に、内定の電話がきた
  3. 働き始めて1週間。「続ける」ために私が決めたこと
  4. もう二度と倒れないために、自分と約束したルール
  5. あの頃の私と同じ場所にいる人に、これだけは言いたい

10. 撮影・表現（顔出しなしの作り方）

顔出しなしはハンデではなく、むしろ最強の共感装置。顔がない＝視聴者が「自分」を重ねられる。声（ナレ）とテロップが主役。

  • 画づくり：主観カット（本人の目線）／後ろ姿・足元・手元・横顔の一部・シルエット・逆光
  • 小物で感情を語る：冷めたコーヒー・閉じたカーテン・点いたままのPC・鳴り続けるアラーム
  • B-roll＋モノローグ：電車・空・キーボード＋本音のひとこと
  • 節目は二度撮れない：診断・退職・通所初日・実習・内定は“起きた瞬間”に必ず撮影。作り溜め前提で動く

11. 投稿フォーマット＆頻度

フォーマット

  • 縦動画（30〜60秒）：掴み3秒（結論・本音）→ その回の一歩 or つまずき → 本音 → 次回引き → フォロー/保存誘導
  • カルーセル：表紙（状態の一言）→ 3〜7枚（出来事と心情）→ 小さな一歩 or 問い → 保存/面談誘導

頻度（媒体で分ける）

Instagram（連載・縦動画／外注編集）

週1〜2本。作り込み重視のアンカー

Threads（ベタ打ち短文／内製）

可能なら日次1〜2本。低コストで共感・拡散、流入の量を稼ぐ

役割：Instagram＝ストックと信頼、Threads＝初動と拡散。両方からプロフィール→面談へ。

12. 集客動線（ゴール＝面談）

SNS投稿（共感・気づき）
　→ プロフィール
　　→ LINE登録（餌＝無料の自己診断チェック／小冊子DL）
　　　→ 無料面談（電話なし・文字／オンラインでOK）← ゴール
　　　　→ 入校 → 社会復帰 → 本人を次の連載へ（永久ループ）

  • 電話はこの層の最大のハードル。だから「話すのが苦手でも大丈夫。まずは文字で」を前面に。第一接点はLINEチャット、電話は選択制。これが面談CVRを大きく左右する
  • CTAは毎回1つだけ。ゴールは一貫して「LINE登録 → 面談予約」。「まず話を聞くだけでいい」でハードルを下げる
  • 連載＝信頼を作りフォローさせる／知識・チェックリスト＝保存と流入／不安解消＝面談の背中を押す
  • 全国オンライン＝「どこに住んでても在宅で学べる」を強みとして毎回効かせる

13. KPI・スケジュール

最重要KPI＝面談予約数。次点：保存率・プロフィール遷移率・LINE登録数。

時期やること目安
Week 0主役確定＋同意書／アカウント開設／撮影スタート—
Month 1連載1〜3話＋単発で“伸びる型”を特定保存率・遷移率を見る
Month 2連載継続＋LINE導線を稼働。面談発生を計測開始面談が週単位で発生
Month 3節目回（スクール・変化）で伸ばす面談が定常化する土台

14. 法務・リスク管理

薬機法・医療広告ガイドライン・障害福祉サービス広告に配慮。最初からルール化する。

NG

「治る／必ず良くなる」断定／体験談を治療効果として見せる／エビデンスなし数字／「これに当てはまればうつ」の断定

言い換え

治る→対処できる／うつです→受診の目安かも／数字→定性表現／ストーリーは「利用者の歩み」として

  • 撮影・公開同意書：顔出し範囲を本人が選べる形（顔／モザイク／後ろ姿／声のみ）。医療情報（診断名・服薬）は本人許可分のみ
  • 撤回権を最初に約束：「嫌になったらいつでも消せる」＝信頼の土台。他者の映り込みは本人にも同意。再現映像は「イメージ」表記
  • コメント方針：医療相談には答えず受診を促す定型／自傷示唆・アンチ対応・NGワード一覧を開始前に整備
  • 連載リスク：主役の体調が生命線。卒業生の振り返りを併走、または主役2人体制でリスク分散
  • 運用共通：数字・断言の前に裏取り／文字数は実測／過去テーマと被り照合／AI口調NG

15. 美術・ロケ設定（部屋の作り込み）

顔出しなし＝部屋そのものが主役。光と小物で感情を語る。リアルさ（作り物っぽくない）が最優先で、“お金かかってる感”はNG。ベースはレンタルスペース（例：13㎡の個室・フローリング・ベランダ付き）を想定。

あかりの部屋＝「転落型」（昔の名残 × 今の荒れ）

  • 散らかり＝回復のバロメーター：Phase3（どん底）で最も散らかり、Phase6〜7で少しずつ整う
